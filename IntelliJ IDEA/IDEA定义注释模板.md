# IDEA定义注释模板

配置路径：

```
File —> settings —> Editor —> File and Code Templates —> Files
```

*配置示例：*

```java
/**
 * <p></p>
 *
 * @author ${USER}
 * @version ${VERSION}
 * @since ${DATE} ${TIME}
 */
public class ${NAME} {
}
```
