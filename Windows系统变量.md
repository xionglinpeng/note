# Windows系统变量

```shell
C:\Users\chinasoft.lp.xiong> echo %WINDIR%

```

```shell
C:\Users\chinasoft.lp.xiong> echo %SYSTEMROOT%

```

```shell
C:\Users\chinasoft.lp.xiong> echo %SYSTEMDRIVE%
```
